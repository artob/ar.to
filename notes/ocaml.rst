Installation
============

Mac OS X with MacPorts
----------------------

* ``sudo port install ocaml``
* ``sudo port install opam``
* ``opam init``
* ``opam switch 4.02.3``

Ubuntu 14.04
------------

* ``apt-add-repository ppa:avsm/ppa``
* ``aptitude update``
* ``aptitude install ocaml-nox opam``
* ``opam init``
* ``opam switch 4.02.3``
